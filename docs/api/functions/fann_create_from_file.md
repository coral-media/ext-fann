
Loads a neural network from a saved configuration file.

***

* Full name: `fann_create_from_file`
* Defined in: `fann.stub.php`

## Parameters

| Parameter             | Type       | Description            |
|-----------------------|------------|------------------------|
| `$configuration_file` | **string** | Path to a `.net` file. |

## Return Value

**resource|false**

Neural network resource on success, false on failure.
