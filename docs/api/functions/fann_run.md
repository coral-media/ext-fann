
Runs inference for one input vector.

***

* Full name: `fann_run`
* Defined in: `fann.stub.php`

## Parameters

| Parameter | Type         | Description                                        |
|-----------|--------------|----------------------------------------------------|
| `$ann`    | **resource** | Neural network resource.                           |
| `$input`  | **array**    | Input values; size must match network input count. |

## Return Value

**array|false**

Output vector on success, false on failure.
