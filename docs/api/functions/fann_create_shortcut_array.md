***

* Full name: `fann_create_shortcut_array`
* Defined in: `fann.stub.php`

## Parameters

| Parameter | Type      | Description                       |
|-----------|-----------|-----------------------------------|
| `$layers` | **array** | Number of neurons for each layer. |

## Return Value

**resource|false**

Neural network resource on success, false on failure.
