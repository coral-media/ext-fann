
Allocates an empty training-data structure.

***

* Full name: `fann_create_train`
* Defined in: `fann.stub.php`

## Parameters

| Parameter     | Type    | Description                      |
|---------------|---------|----------------------------------|
| `$num_data`   | **int** | Number of training rows.         |
| `$num_input`  | **int** | Number of input values per row.  |
| `$num_output` | **int** | Number of output values per row. |

## Return Value

**resource|false**

Training data resource on success, false on failure.
